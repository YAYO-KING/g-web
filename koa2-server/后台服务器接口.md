安装koa  koa-router
安装mongoose 连接mongodb数据库
安装koa-bodyparser 解析post请求中的body参数
npm i koa koa-bodyparser koa-router mongoose -S

安装nodemon，实现热加载
npm i nodemon -S

安装jsonwebtoken，用来给登录接口生成token使用
npm i jsonwebtoken -S
安装koa-jwt,koa设置jwt使用
npm i koa-jwt -S

安装koa-body，用来上传文件
npm install koa-body --save
